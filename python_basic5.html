Quotes — Single, Double, Triple
Three ways to write a string literal
01

✨
f-strings (f'...')
Embed variables directly inside a string — Python 3.6+
02
name = 'Alice' age = 30 # Embed variables with { } s = f'My name is {name} and I am {age}' # → 'My name is Alice and I am 30' # Expressions work too s2 = f'Next year I will be {age + 1}' # → 'Next year I will be 31'
🔗
str() + Concatenation
Convert other types to string, and join strings together
03
# Convert any type to string s = str(42) # → '42' # Concatenate with + first = 'Hello' last = 'World' full = first + ' ' + last # → 'Hello World' # Repeat with * line = '-' * 20 # → '--------------------'
