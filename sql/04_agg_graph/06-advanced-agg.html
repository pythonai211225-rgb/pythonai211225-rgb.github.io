Conditional UPDATE with CASE

Use CASE inside UPDATE to apply different formulas to different rows in one statement.

Query — tiered bonus
UPDATE sales
SET bonus =
  CASE
    WHEN product = 'Laptop'  THEN amount * 0.15
    WHEN amount IS NOT NULL   THEN amount * 0.08
    ELSE 0
  END;

COALESCE — replace NULL totals with 0

When a group's SUM is NULL (all values in the group are NULL), COALESCE replaces it with a fallback.

Query 1
SELECT
  rep_name,
  product,
  COALESCE(SUM(amount), 0) AS revenue
FROM sales
GROUP BY rep_name, product
ORDER BY rep_name, product;

ROUND — clean up AVG results

AVG often produces long decimals. ROUND(value, n) trims them to n decimal places.

Query 2
SELECT
  product,
  ROUND(AVG(amount), 2) AS avg_price,
  ROUND(SUM(amount), 2) AS total_revenue
FROM sales
GROUP BY product
ORDER BY total_revenue DESC;

COUNT(CASE WHEN …) — conditional counting

Count only rows matching a condition, without filtering out others.

Query 3
SELECT
  rep_name,
  COUNT(*) AS total_sales,
  COUNT(CASE WHEN product = 'Laptop' THEN 1 END) AS laptop_sales,
  COUNT(CASE WHEN amount > 1000   THEN 1 END) AS big_sales
FROM sales
GROUP BY rep_name;

CASE returns 1 when true, NULL otherwise. COUNT ignores NULLs — so it effectively counts only the true cases.

Common mistakes

1

Selecting a non-grouped column

Every column in SELECT must either be in GROUP BY or wrapped in an aggregate.

Wrong
-- sale_date not in GROUP BY
SELECT rep_name, sale_date,
       SUM(amount)
FROM sales
GROUP BY rep_name;
Fixed ✓
SELECT rep_name,
       SUM(amount) AS total
FROM sales
GROUP BY rep_name;
2

WHERE on an aggregate

WHERE runs before GROUP BY — the aggregate result doesn't exist yet.

Wrong
-- can't use SUM() in WHERE
SELECT rep_name, SUM(amount)
FROM sales
GROUP BY rep_name
WHERE SUM(amount) > 1500;
Fixed ✓
SELECT rep_name, SUM(amount)
FROM sales
GROUP BY rep_name
HAVING SUM(amount) > 1500;

Putting it all together — full report

Query 4
SELECT
  rep_name,
  COUNT(*) AS total_sales,
  COUNT(CASE WHEN product = 'Laptop' THEN 1 END) AS laptops,
  COALESCE(ROUND(SUM(amount), 2), 0) AS revenue,
  COALESCE(ROUND(AVG(amount), 2), 0) AS avg_sale
FROM sales
GROUP BY rep_name
HAVING COUNT(*) >= 2
ORDER BY revenue DESC;
