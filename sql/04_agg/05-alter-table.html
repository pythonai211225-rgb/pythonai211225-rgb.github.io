Topic 5

ALTER TABLE — Add Columns & Fill Data

ALTER TABLE lets you change an existing table's structure. ADD COLUMN adds a new field — every existing row gets NULL in that column until you fill it with UPDATE.

ADD COLUMN basics

Add a bonus column to the existing sales table. All current rows will have NULL in the new column.

Syntax
ALTER TABLE sales
ADD COLUMN bonus REAL;

After this statement the table has 7 columns. Every existing row has NULL for bonus. You cannot drop a column in SQLite — only add one.

Fill the new column with UPDATE

Set bonus to 10 % of amount. Rows where amount is NULL will still have NULL bonus.

Query 1 — flat percentage
UPDATE sales
SET    bonus = amount * 0.10;

Conditional UPDATE with CASE

Give a higher bonus to Laptop sales and a lower one to everything else.

Query 2 — tiered bonus
UPDATE sales
SET bonus =
  CASE
    WHEN product = 'Laptop' THEN amount * 0.15
    WHEN amount IS NOT NULL  THEN amount * 0.08
    ELSE 0
  END;

ADD COLUMN with a DEFAULT

SQLite lets you specify a default so new rows automatically get the value — and existing rows also receive it immediately.

Query 3
ALTER TABLE sales
ADD COLUMN is_flagged INTEGER DEFAULT 0;

All 10 existing rows now have is_flagged = 0. Future INSERTs that omit is_flagged also default to 0.
