Netflix Practice Lab

Use a real CSV import and run practical filtering and ranking queries.

Fun Twist

Import Netflix titles CSV to SQLite and run practical queries on real data.

  1. Go to dataset page: Netflix Movies and TV Shows (Kaggle)
  2. Click Download and save the ZIP file
  3. Extract the ZIP and find the CSV file

Open CSV in DB Browser and convert to .db

  1. Open DB Browser for SQLite
  2. Click New Database and save a file, for example: netflix.db
  3. Go to File > Import > Table from CSV file
  4. Select the CSV you downloaded from Kaggle
  5. Set table name to netflix_movies_detailed_up_to_2025 and confirm first row is column names
  6. Click OK, then press Write Changes so data is saved into the .db file
Target query shape
SELECT DISTINCT type
FROM netflix_movies_detailed_up_to_2025
ORDER BY type;
Category search example
SELECT title, release_year, rating
FROM netflix_movies_detailed_up_to_2025
WHERE type = 'Movie'
  AND genres LIKE '%Action%'
  AND release_year >= 2015
ORDER BY rating DESC
LIMIT 10;
In this CSV, duration is empty in all rows, so use genres, release_year, rating, vote_count, and cast for filtering
Search queries to try
-- 1) What type values exist in the dataset?
SELECT DISTINCT type
FROM netflix_movies_detailed_up_to_2025
ORDER BY type;

-- 2) Distinct cast entries for action movies (cast text blocks)
SELECT DISTINCT "cast"
FROM netflix_movies_detailed_up_to_2025
WHERE type = 'Movie'
  AND genres LIKE '%Action%'
  AND "cast" IS NOT NULL
  AND TRIM("cast") != ''
LIMIT 20;

-- 3) Action movies from 2015 and later
SELECT title, release_year, rating
FROM netflix_movies_detailed_up_to_2025
WHERE type = 'Movie'
  AND genres LIKE '%Action%'
  AND release_year >= 2015
ORDER BY release_year DESC
LIMIT 20;

-- 4) Top 10 comedy titles by rating (with enough votes)
SELECT title, release_year, rating, vote_count
FROM netflix_movies_detailed_up_to_2025
WHERE type = 'Movie'
  AND genres LIKE '%Comedy%'
  AND vote_count >= 100
ORDER BY rating DESC, vote_count DESC
LIMIT 10;

-- 5) Drama titles where cast is missing
SELECT title, release_year
FROM netflix_movies_detailed_up_to_2025
WHERE type = 'Movie'
  AND genres LIKE '%Drama%'
  AND ("cast" IS NULL OR TRIM("cast") = '');
