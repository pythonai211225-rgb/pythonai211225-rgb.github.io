Topic 5

DISTINCT for Unique Values

Use DISTINCT when you want unique values only, without duplicates.

Source table

We will use this movies table for DISTINCT examples.

id title genre year language
1 Metro Rush Action 2005 English
2 Quiet Lake Drama 2012 English
3 Night Pulse Horror 2018 Spanish
4 Skyline Code Action 2021 English
5 Tiny Planet Family 2010 French
6 Silver Track Thriller 2016 Spanish
7 Golden Path Drama 2016 French

DISTINCT on one column

Query 1
-- unique genres
SELECT DISTINCT genre
FROM movies
ORDER BY genre ASC;

Query 2
-- unique release years

DISTINCT on multiple columns

-- unique genre-language pairs
SELECT DISTINCT genre, language
FROM movies
