Topic 2

LIKE, GLOB, BETWEEN, IN, IS NULL

This is your filter toolbox for pattern matching, ranges, lists, and missing values.

Setup table

Setup
CREATE TABLE products (
  id INTEGER PRIMARY KEY,
  product_name TEXT NOT NULL,
  category TEXT NOT NULL,
  price REAL NOT NULL,
  stock INTEGER
);

INSERT INTO products VALUES

Filter patterns and ranges

Read each query, then compare it with the result table below it.

Query 1
-- LIKE: names starting with 'M'
SELECT product_name FROM products
    WHERE product_name LIKE 'M%';
product_name
Mouse Pro
Monitor 27
Mini PC
Query 2
-- GLOB: exactly 5 letters then space then anything
SELECT product_name FROM products
    WHERE product_name GLOB '????? *';
product_name
Mouse Pro
Query 3
-- BETWEEN: inclusive range
SELECT product_name, price FROM products
    WHERE price BETWEEN 40 AND 300;
product_name price
Mouse Pro 49.0
Monitor 27 289.0
Dock Station 129.0
Query 4
-- IN: match one of many categories
SELECT product_name, category FROM products
    WHERE category IN ('Accessories', 'Computers');

Query 5
-- IS NULL: unknown or missing stock
SELECT product_name FROM products
WHERE stock IS NULL;

Exercise

Use every filter type at least once

  1. Find all products with name ending in "Pro" (LIKE)
  2. Find products priced between 30 and 150 (BETWEEN)
  3. Find products in categories Computers or Displays (IN)
  4. Find products with known stock (not null)
  5. Bonus: use GLOB to match names that start with "M" and have a space
Tip: BETWEEN includes both border values
