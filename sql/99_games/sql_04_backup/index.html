SQL Stair Sprint

Start on floor 0 and climb by reaching the staircase. Each floor has a different maze shape and checkpoint positions. A wrong answer sends you back to that floor start.

Checkpoints

Easy x3: SELECT basics
Medium x2: JOIN / GROUP BY
Hard x1: subqueries / NOT EXISTS

Floor: 0

State: Reach the staircase

Position: Start

Progress: E 0/3 | M 0/2 | H 0/1

Staircase: Always open

Question Bank: 15 total

Move with Arrow keys or WASD.

When a checkpoint question appears, click an answer or use keys 1-4.
