Famous SQL Engines

There's one SQL language, but many different database engines that run it. Each has a different personality — some are built for power, some for simplicity, some for scale

All speak SQL Different strengths We'll use SQLite in this course
The big five

SQL engines you'll encounter in the wild

All five speak SQL. The syntax is 90% identical across them — skills you learn in one transfer to the others. The differences are in features, scale, licensing, and deployment model

🐘

PostgreSQL

Free & open source

The most feature-rich open-source database. Supports JSON, full-text search, geospatial data, custom types, and extensions. Often called "the world's most advanced open source database." Trusted by Apple, Reddit, Spotify, and Instagram

Choose it when you want power, reliability, and rich features. The default choice for serious production apps
🐬

MySQL / MariaDB

Free & open source

The most widely deployed open-source database in the world. Powers WordPress, Shopify, Twitter (historically), Facebook (historically). MySQL was bought by Oracle; MariaDB is the community fork that stays truly open. Very fast for read-heavy workloads

Choose it when you're building a web app (especially with PHP/WordPress) or need maximum compatibility with hosting providers
🪶

SQLite

Free · Public domain

The most deployed database engine on Earth — it lives inside your app as a library, with no server. Every Android phone, every iPhone, every Chrome browser, every Firefox installation has SQLite running inside it. The entire database is a single file

Choose it when you're learning SQL, building a mobile app, a desktop app, or need embedded storage without a server
🪟

SQL Server

Paid · Microsoft

Microsoft's flagship database, tightly integrated with the Windows ecosystem, .NET, and Azure. Has a free Express edition for small apps. Common in corporate environments, government, and enterprises already using Microsoft stack. Excellent tooling with SQL Server Management Studio

Choose it when your company runs on Microsoft technologies and Azure, or needs deep Windows integration
🏛️

Oracle Database

Paid · Enterprise

The old giant. Oracle has been in the database business since 1979 and still dominates banking, telecommunications, and large government systems. Massively powerful and feature-rich, but also massively expensive. Licensing can cost tens of thousands of dollars per CPU core

Choose it when you work at a large enterprise or bank that has already invested in the Oracle ecosystem and needs its specific enterprise features

Why SQLite for this course

🪶 SQLite is the perfect learning database

  • ✓Zero setup — no server to install, configure, or start. Download one file and you're running
  • ✓The SQL you learn here is real SQL — 95% of what you write in SQLite works identically in PostgreSQL and MySQL
  • ✓Built into Python — import sqlite3 is in the standard library. No package to install
  • ✓Used everywhere — SQLite is not a toy. It is deployed in more places than any other database engine on the planet
  • ✓One file = one database — easy to copy, share, delete, and start fresh while learning
