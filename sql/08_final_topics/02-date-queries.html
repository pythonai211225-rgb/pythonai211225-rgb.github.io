Date Queries — Filter by Time

SQLite stores dates as text. strftime() lets you pull out any piece — year, month, day, hour — and filter on it.

strftime() Cheatsheet

%Y
4-digit year
→ '2026'
%m
Month (zero-padded)
→ '03'
%d
Day of month
→ '15'
%H
Hour (24h)
→ '14'
%M
Minutes
→ '32'
%S
Seconds
→ '05'
%Y-%m-%d
Date only (no time)
→ '2026-03-15'
%w
Day of week (0=Sun)
→ '1' (Monday)

strftime always returns a TEXT string. To compare numbers wrap it in CAST(strftime('%H', col) AS INTEGER).

Filter by Year

Query
SELECT customer, amount, created_at
FROM   orders
WHERE  strftime('%Y', created_at) = '2026';

All 10 rows — all orders are in 2026. Change '2026' to '2025' and you'd get 0 rows.

Filter by Month

Query — all March orders
SELECT customer, amount, created_at
FROM   orders
WHERE  strftime('%m', created_at) = '03';

Month is always zero-padded: March = '03', not '3'. Same for days and hours.

Filter by Specific Day

Query — March 15th only
SELECT customer, amount, created_at
FROM   orders
WHERE  DATE(created_at) = '2026-03-15';

DATE(created_at) strips the time portion and returns just '2026-03-15'. Simpler than using strftime('%Y-%m-%d') for exact-day comparisons.

Combination — March 15th Between 2–4 PM

Combine multiple date conditions with AND. Use CAST to compare the hour as an integer for BETWEEN.

Query
SELECT customer, amount, created_at
FROM   orders
WHERE  DATE(created_at)                          = '2026-03-15'
  AND  CAST(strftime('%H', created_at) AS INTEGER) BETWEEN 14 AND 15;
customeramountcreated_at
Felix185.002026-03-15 15:22:00
Greta220.002026-03-15 14:10:00

💡 BETWEEN 14 AND 15 includes hours 14 and 15 (2 PM and 3 PM). Emma's 09:00 order on March 15 is excluded because it's in the morning.

Bonus — Group Revenue by Month

strftime works in GROUP BY too — slice your data into any time bucket.

Query
SELECT   strftime('%Y-%m', created_at) AS month,
         COUNT(*) AS orders,
         ROUND(SUM(amount), 2)  AS revenue
FROM     orders
GROUP BY month
ORDER BY month;

Exercise

Query the orders table by date

  1. Find all orders placed on a Monday — use strftime('%w', created_at) = '1' (Monday = 1 in SQLite)
  2. Find all orders placed in Q1 (January, February, March) — use strftime('%m') with BETWEEN or IN
