1. UNION — Stacking Results

Combine rows from two SELECT queries into one result. Learn when to use UNION (deduplicate) vs UNION ALL (keep everything), plus INTERSECT and EXCEPT.

2. Date Queries — Time-Based Filtering

Use strftime to query by year, month, day, or hour. Combine conditions to find orders on a specific date between specific hours.

3. Triggers

Triggers fire SQL automatically on INSERT, UPDATE, or DELETE. Use AFTER to sync summary tables, BEFORE to validate and reject bad data.

CREATE TRIGGER · AFTER INSERT · BEFORE · NEW / OLD

4. Views

A view is a saved SELECT stored in the database. Query it like a table — wrap complex JOINs, filter with WHERE, or add CASE columns, all in one definition.

CREATE VIEW · DROP VIEW · CASE · Always Fresh
