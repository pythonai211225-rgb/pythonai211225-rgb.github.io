Topic 5

WHERE vs HAVING

Use WHERE for row-level filters before grouping, HAVING for group-level filters after aggregation.

Execution flow (simplified)

  1. FROM
  2. WHERE
  3. GROUP BY
  4. HAVING
  5. SELECT
  6. ORDER BY

Correct pattern using both

Query 1
SELECT
  region,
  COUNT(*) AS orders_count,
  SUM(amount) AS total_sales
FROM sales
WHERE order_date >= '2025-01-01'
GROUP BY region
HAVING SUM(amount) >= 5000
ORDER BY total_sales DESC;
WHERE limits rows first. HAVING then removes groups that do not meet aggregated conditions.

Wrong vs right

Wrong

SELECT region, SUM(amount)
FROM sales
WHERE SUM(amount) > 5000
GROUP BY region;

Right

SELECT region, SUM(amount) AS total_sales
FROM sales
GROUP BY region
HAVING SUM(amount) > 5000;
