Topic 2

SUM, AVG, MIN, MAX with NULLs

Understand NULL behavior in aggregates, then fix reports with COALESCE and ROUND.

Aggregate behavior with NULL

Query 1
SELECT
  SUM(salary) AS sum_salary,
  AVG(salary) AS avg_salary,
  MIN(salary) AS min_salary,
  MAX(salary) AS max_salary
FROM payroll;
SUM/AVG/MIN/MAX ignore NULL input values. If all rows are NULL, result is NULL.

COALESCE and ROUND for cleaner output

Query 2
SELECT
  department,
  SUM(COALESCE(bonus, 0)) AS total_bonus,
  ROUND(AVG(salary), 2) AS avg_salary_2dp
FROM payroll
GROUP BY department
ORDER BY total_bonus DESC;
Query 3
-- force 0 instead of NULL for an all-NULL aggregate scenario
SELECT
  COALESCE(SUM(bonus), 0) AS safe_bonus_total
FROM payroll
WHERE department = 'Legal';
