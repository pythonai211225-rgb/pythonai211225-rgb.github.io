Topic 6

ALTER TABLE and ADD COLUMN

Change schema safely, then populate new columns with UPDATE statements.

Add a new column

Query 1
ALTER TABLE products
ADD COLUMN stock_quantity INTEGER DEFAULT 0;
Existing rows receive the default value if provided. Without default, they get NULL.

Backfill the new column

Query 2
UPDATE products
SET stock_quantity = 25
WHERE category = 'Accessories';
Query 3
UPDATE products
SET stock_quantity = 100
WHERE stock_quantity IS NULL;

Useful pattern after schema change

Query 4
SELECT
  category,
  SUM(COALESCE(stock_quantity, 0)) AS total_stock
FROM products
GROUP BY category
ORDER BY total_stock DESC;
