Topic 1

COUNT(*) vs COUNT(column) and DISTINCT

Count total rows, count non-NULL values, and count unique values correctly.

Source table

We will use this orders table.

id customer_id coupon_code total_amount
1101WINTER10120.00
2102NULL89.00
3101WINTER1076.00
4103NULL45.00
5104VIP5200.00

COUNT(*) vs COUNT(column)

Query 1
SELECT
  COUNT(*) AS all_rows,
  COUNT(coupon_code) AS coupon_non_null_rows
FROM orders;
COUNT(*) counts every row. COUNT(column) ignores NULLs in that column.

COUNT with DISTINCT

Query 2
SELECT
  COUNT(DISTINCT customer_id) AS unique_customers,
  COUNT(DISTINCT coupon_code) AS unique_coupons
FROM orders;
Query 3
-- list unique coupon values only
SELECT DISTINCT coupon_code
FROM orders
ORDER BY coupon_code;
