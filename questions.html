Question 1

Grade Input Program

Write a program that inputs grades from the user until the user enters -999

Requirements

  • Use a list to store all valid grades
  • If the grade is below 0 or above 100, ignore it
  • If the user enters an invalid value such as a string, ignore it
    • Hint: use try-except
  • At the end, use:
    • sum() and len() to calculate the class average
    • max() to find the maximum grade
  • Print:
    • the class average
    • the maximum grade
    • how many valid students' grades were entered
  • If the user enters -999 before at least 10 valid grades were entered, continue asking for input
  • The student must enter at least 50% valid grades (i.e., at least 10 valid inputs) before the program can finish

Example

Enter grade: 85
Enter grade: 90
Enter grade: abc
Invalid input, skip
Enter grade: 110
Not in range, skip
Enter grade: -5
Not in range, skip
Enter grade: 72
Enter grade: 60
Enter grade: -999
Need at least 10 valid grades. Keep entering.
Enter grade: 78
Enter grade: 95
Enter grade: 88
Enter grade: 55
Enter grade: 91
Enter grade: -999
Need at least 10 valid grades. Keep entering.
Enter grade: 67
Enter grade: 74
Enter grade: 83
Enter grade: -999

Output

Number of valid grades: 12
Class average: 78.17
Highest grade: 95
Question 2

Median of an Unsorted List

Write a function that receives an unsorted list of numbers and returns the median.

The median is the middle value when the list is sorted.

If the list has an odd number of elements, the median is the middle element.

If the list has an even number of elements, the median is the average of the two middle elements.

Function Signature

def find_median(numbers: list) -> float:

Requirements

  • Solve this question as a function
  • Do not assume the list is sorted — sort it inside the function
  • Return the median as a float

Examples

Input:

[3, 1, 4, 1, 5]

Sorted: [1, 1, 3, 4, 5] — middle element is index 2

Output:

3.0

Input:

[7, 2, 10, 9]

Sorted: [2, 7, 9, 10] — two middle elements are 7 and 9, average = (7 + 9) / 2

Output:

8.0

Input:

[42]

Sorted: [42] — only one element

Output:

42.0

Duplicate String Detector
