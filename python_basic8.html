01 — creation
Ways to Create a Dict
{ }
{ } Curly Braces
Most common way — key: value pairs separated by commas
01
# String keys person = { 'name': 'Alice', 'age': 30, 'city': 'Tel Aviv' } # Empty dict empty = {}
🏗️
dict() Constructor
Use keyword arguments or a list of pairs
02
# Keyword arguments d = dict(name='Bob', age=25) # From a list of (key, value) pairs pairs = [('a', 1), ('b', 2), ('c', 3)] d2 = dict(pairs) # From two lists using zip keys = ['x', 'y', 'z'] vals = [10, 20, 30] d3 = dict(zip(keys, vals))
⚡
Dict Comprehension
Build a dict from an expression in one line
03
# Square of each number squares = {x: x**2 for x in range(1, 6)} # → {1:1, 2:4, 3:9, 4:16, 5:25} # Invert a dict (swap keys and values) original = {'a': 1, 'b': 2, 'c': 3} inverted = {v: k for k, v in original.items()} # → {1:'a', 2:'b', 3:'c'}
📋
dict.fromkeys()
Create a dict from a list of keys with a default value
04
keys = ['a', 'b', 'c'] # All values set to 0 d = dict.fromkeys(keys, 0) # → {'a': 0, 'b': 0, 'c': 0} # Default value is None if not specified d2 = dict.fromkeys(keys) # → {'a': None, 'b': None, 'c': None}

.popitem()

LIVE DEMO
d = {'x': 10, 'y': 20, 'z': 30} # Removes and returns the last pair item = d.popitem() print("removed:", item) # → ('z', 30) print(d) # → {'x': 10, 'y': 20}

🧩
.setdefault(key, default)
Get value if key exists — insert default if it doesn't
LIVE DEMO
d = {'hits': 5} # Key exists — returns existing value, no change print(d.setdefault('hits', 0)) # → 5 # Key missing — inserts 0 and returns it print(d.setdefault('misses', 0)) # → 0 print(d)

🔎
in / not in
Check if a KEY exists — very fast O(1) lookup
LIVE DEMO
d = {'name': 'Alice', 'age': 30} # Check key presence print('name' in d) # → True print('phone' not in d) # → True # in checks KEYS only, not values print('Alice' in d) # → False
>>>click ▶ RUN

LIVE DEMO
d = {'a': 1, 'b': 2, 'c': 3} # Loop over keys (default) for k in d: print(k) # Loop over key + value pairs for k, v in d.items(): print(k, '→', v)

📋
.copy() & .clear()
Shallow copy a dict, or empty it entirely
LIVE DEMO
original = {'a': 1, 'b': 2} # Shallow copy — changes don't affect original clone = original.copy() clone['c'] = 3 print(original) # → {'a': 1, 'b': 2} # Clear removes all keys clone.clear() print(clone) # → {}
