Python Sets

# unordered · no duplicates · mutable · fast lookup

01 — creation
Ways to Create a Set
{ }
{ } Curly Braces
Most direct way — duplicates are removed automatically
01
# Basic set — no duplicates allowed fruits = {'apple', 'banana', 'cherry'} # Duplicates are silently dropped nums = {1, 2, 2, 3, 3, 3} # → {1, 2, 3} # Mixed types are fine mixed = {1, 'hello', True, 3.14}
🏗️
set() Constructor
Convert any iterable — also used to create an empty set
02
# From a list (removes duplicates) from_list = set([1, 2, 2, 3]) # → {1, 2, 3} # From a string (unique characters) from_str = set('hello') # → {'h', 'e', 'l', 'o'} # Empty set — must use set(), NOT {} empty = set() # {} creates a dict, not a set!
⚡
Set Comprehension
Build a set from an expression — like list comprehension
03
# Squares — duplicates dropped automatically squares = {x**2 for x in range(6)} # → {0, 1, 4, 9, 16, 25} # Unique first letters from a list words = ['cat', 'car', 'dog', 'deer'] initials = {w[0] for w in words} # → {'c', 'd'}
🚫
frozenset()
An immutable set — cannot be changed after creation
04
# frozenset is like a set but immutable fs = frozenset([1, 2, 3, 2, 1]) # → frozenset({1, 2, 3}) # Can be used as a dict key (regular sets cannot) lookup = {frozenset({1, 2}): 'pair'} # Trying to add raises an error # fs.add(4) → AttributeError
