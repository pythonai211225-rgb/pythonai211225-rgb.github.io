Python Tuples

# ordered · immutable · allow duplicates · zero-indexed

🔒 immutable — once created, a tuple cannot be changed
01 — creation
Ways to Create a Tuple
( )
( ) Parentheses
Most readable way — the comma is what makes a tuple, not the parens
01
# Standard tuple colors = ('red', 'green', 'blue') # Mixed types allowed mixed = (42, 'hello', True, 3.14) # Empty tuple empty = () # Nested tuple nested = ((1, 2), (3, 4))
⚠️
Single-element Tuple
You MUST add a trailing comma — otherwise it's just a value in parens
02
# This is NOT a tuple — it's just the number 42 not_tuple = (42) print(type(not_tuple)) # → <class 'int'> # THIS is a tuple — notice the trailing comma one_tuple = (42,) print(type(one_tuple)) # → <class 'tuple'>
🏗️
tuple() Constructor
Convert any iterable into a tuple
03
# From a list from_list = tuple([1, 2, 3]) # → (1, 2, 3) # From a string from_str = tuple('hello') # → ('h', 'e', 'l', 'l', 'o') # From a range from_range = tuple(range(1, 6)) # → (1, 2, 3, 4, 5)
📦
Packing & Unpacking
Assign multiple values at once — Python's most elegant feature
04
# Packing — no parens needed point = 10, 20, 30 # → (10, 20, 30) # Unpacking — assign each value to a variable x, y, z = point print(x, y, z) # → 10 20 30 # Swap two variables in one line a, b = 1, 2 a, b = b, a # → a=2, b=1
✳️
Star Unpacking
Capture the "rest" of a tuple with * operator
05
t = (1, 2, 3, 4, 5) # First item, rest goes into a list first, *rest = t print(first) # → 1 print(rest) # → [2, 3, 4, 5] # First, middle, and last head, *mid, tail = t # → head=1 mid=[2,3,4] tail=5
02 — access
Indexing & Slicing
🎯
Index Access
Access individual items by position — positive and negative
LIVE DEMO
t = ('a', 'b', 'c', 'd', 'e') # Positive index — from the front print(t[0]) # → 'a' # Negative index — from the back print(t[-1]) # → 'e'

✂️
Slicing
Extract a sub-tuple with [start : stop : step]
LIVE DEMO
t = (0, 1, 2, 3, 4, 5) # Items at index 1, 2, 3 print(t[1:4]) # → (1, 2, 3) # Every second item print(t[::2]) # → (0, 2, 4) # Reversed print(t[::-1]) # → (5, 4, 3, 2, 1, 0)

Immutability in Action
You cannot change a tuple — but you can work around it
LIVE DEMO
t = (1, 2, 3) # This would raise a TypeError: # t[0] = 99 → TypeError! # Workaround: convert to list, edit, convert back lst = list(t) lst[0] = 99 t = tuple(lst) print(t) # → (99, 2, 3)
>>>click ▶ RUN
🏷️
namedtuple
Tuples with named fields — access by name or index
LIVE DEMO
from collections import namedtuple # Define a namedtuple type Point = namedtuple('Point', ['x', 'y', 'z']) # Create an instance p = Point(10, 20, 30) # Access by name print(p.x, p.y, p.z) # → 10 20 30 # Access by index still works print(p[0]) # → 10
