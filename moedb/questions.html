Question 1

Rank Input Program

Write a program that inputs ranks from the user until the user enters -999

Requirements

  • Use a list to store all valid ranks
  • If the rank is below 1 or above 5, ignore it
  • If the user enters an invalid value such as a string, ignore it
    • Hint: use try-except
  • At the end, use:
    • sum() and len() to calculate the average rank
    • max() to find the highest rank
  • Print:
    • the average rank
    • the highest rank
    • how many valid ranks were entered
  • If the user enters -999 before at least 10 valid ranks were entered, continue asking for input
  • The student must enter at least 50% valid grades (i.e., at least 10 valid inputs) before the program can finish

Example

Enter rank: 5
Enter rank: 4
Enter rank: abc
Invalid input, skip
Enter rank: 8
Not in range, skip
Enter rank: 0
Not in range, skip
Enter rank: 3
Enter rank: 2
Enter rank: -999
Need at least 10 valid ranks. Keep entering.
Enter rank: 5
Enter rank: 4
Enter rank: 3
Enter rank: 1
Enter rank: 5
Enter rank: -999
Need at least 10 valid ranks. Keep entering.
Enter rank: 2
Enter rank: 4
Enter rank: 5
Enter rank: -999

Output

Number of valid ranks: 12
Average rank: 3.58
Highest rank: 5
Question 2

special_percentile

Write a function that receives a percentile such as 25 and an unsorted list of numbers, sorts the list, and returns the number at that percentile.

First sort the list.

Then find the percentile position.

If the position is not a whole number, round down.

For this question, use the exact behavior shown in the examples below.

Function Signature

def special_percentile(percent: int, numbers: list) -> float:

Requirements

  • Solve this question as a function
  • Do not assume the list is sorted — sort it inside the function
  • The function receives the percentile and the list
  • Return the selected number

Examples

Input:

special_percentile(25, [40, 10, 30, 20])

Sorted: [10, 20, 30, 40] — for 25% with 4 numbers, return the 2nd number

Output:

20

Input:

special_percentile(25, [50, 10, 40, 30, 20])

Sorted: [10, 20, 30, 40, 50] — for this question, 25% returns the 2nd number

Output:

20

Input:

special_percentile(50, [9, 1, 7, 3])

Sorted: [1, 3, 7, 9]

Output:

7

Question 4

Reverse String Detector

Write a program that inputs strings from the user until the user enters quit

Requirements

  • After the input ends, check whether one of the entered strings also appeared in reverse
  • For example, if the user entered abc and also cba, that counts as a match
  • If at least one reversed pair exists, print that a reversed match was found
  • Otherwise, print that no reversed match was found

Example 1

Enter string: abc
Enter string: banana
Enter string: cba
Enter string: quit

Output:

Reversed match found

Example 2

Enter string: dog
Enter string: cat
Enter string: bird
Enter string: quit

Output:

No reversed match found
