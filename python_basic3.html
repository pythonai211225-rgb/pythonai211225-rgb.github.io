Python Lists

# ordered · mutable · allow duplicates · zero-indexed

01 — creation
Ways to Create a List
01
[ ] Square Brackets
# Most common way fruits = ['apple', 'banana', 'cherry'] numbers = [1, 2, 3, 4, 5] mixed = [42, 'hello', True, 3.14] empty = []
02
list() Constructor
# From other iterables from_str = list('hello') # → ['h','e','l','l','o'] from_range = list(range(1, 6)) # → [1, 2, 3, 4, 5] from_tuple = list((10, 20, 30))
03
List Comprehension
# Powerful one-liner creation squares = [x**2 for x in range(6)] # → [0, 1, 4, 9, 16, 25] evens = [x for x in range(10) if x%2==0] # → [0, 2, 4, 6, 8]
04
Repeat & Multiply
# Repeat with * operator zeros = [0] * 5 # → [0, 0, 0, 0, 0] pattern = [1, 2] * 3 # → [1, 2, 1, 2, 1, 2] nested = [[0]*3 for _ in range(3)]
05
split() from String
# Split a string into list words = "cat dog fish bird".split() # → ['cat', 'dog', 'fish', 'bird'] csv = "a,b,c,d".split(",") # → ['a', 'b', 'c', 'd']
06
Indexing & Slicing
lst = [10, 20, 30, 40, 50] first = lst[0] # → 10 last = lst[-1] # → 50 slice_ = lst[1:4] # → [20, 30, 40] rev = lst[::-1] # → [50,40,30,20,10]
