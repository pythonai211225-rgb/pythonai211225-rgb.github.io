Python Lists

# ordered · mutable · allow duplicates · zero-indexed

01 — creation
Ways to Create a List
📋
[ ] Square Brackets
Most common and direct way
01
# Most common way fruits = ['apple', 'banana', 'cherry'] numbers = [1, 2, 3, 4, 5] mixed = [42, 'hello', True, 3.14] empty = []
🏗️
list() Constructor
Convert any iterable into a list
02
# From a string from_str = list('hello') # → ['h', 'e', 'l', 'l', 'o'] # From a range from_range = list(range(1, 6)) # → [1, 2, 3, 4, 5] # From a tuple from_tuple = list((10, 20, 30))
⚡
List Comprehension
Build a list from an expression in one line
03
# Squares of 0 to 5 squares = [x**2 for x in range(6)] # → [0, 1, 4, 9, 16, 25] # Only even numbers evens = [x for x in range(10) if x%2==0] # → [0, 2, 4, 6, 8]
✖️
Repeat with * Operator
Repeat a value or pattern to fill a list
04
# Five zeros zeros = [0] * 5 # → [0, 0, 0, 0, 0] # Repeating pattern pattern = [1, 2] * 3 # → [1, 2, 1, 2, 1, 2]
✂️
.split() from String
Split a string into a list of parts
05
# Split by spaces words = "cat dog fish bird".split() # → ['cat', 'dog', 'fish', 'bird'] # Split by comma csv = "a,b,c,d".split(",") # → ['a', 'b', 'c', 'd']
🎯
Indexing & Slicing
Access items by position or range
06
lst = [10, 20, 30, 40, 50] # First item first = lst[0] # → 10 # Last item last = lst[-1] # → 50 # Slice index 1 to 3 part = lst[1:4] # → [20, 30, 40] # Reversed rev = lst[::-1] # → [50, 40, 30, 20, 10]
